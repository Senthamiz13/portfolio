# My Portfolio

This is my personal portfolio website, built with HTML, CSS, and JavaScript. It's designed to be hosted on GitHub Pages.

## Features

- Responsive design
- Smooth scrolling navigation
- Sections for About, Projects, and Contact

## How to View

1. Clone this repository
2. Open `index.html` in your web browser

## Deploying to GitHub Pages

1. Push this repository to GitHub
2. Go to the repository settings
3. Scroll to "Pages" section
4. Select "Deploy from a branch" and choose `main` (or `master`)
5. The site will be available at `https://yourusername.github.io/repository-name`

If the repository is named `yourusername.github.io`, it will be at `https://yourusername.github.io`

## Customization

- Edit `index.html` to update content
- Modify `style.css` for styling changes
- Add functionality in `script.js`
