# Resources Folder

This folder contains all static assets for the portfolio.

## Images

Place your profile image in the `images/` folder:

### Profile Image

- **File name:** `profile.jpg`
- **Location:** `resources/images/profile.jpg`
- **Recommended size:** 200x200px (square)
- **Formats:** JPG, PNG, WebP
- **Tips:** Use a professional headshot photo

**Note:** The path in `config.js` is already set to `resources/images/profile.jpg`. Just add your image file there!

## Organization

```
resources/
├── images/
│   └── profile.jpg    ← Add your profile photo here
└── README.md          ← This file
```
